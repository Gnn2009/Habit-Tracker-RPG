## 🚀 Descripción

**Habit-Tracker-RPG** es una herramienta de terminal que gamifica tu vida diaria. Transforma tus rutinas, el estudio y el ejercicio en misiones de un juego RPG. Al completar tus hábitos diarios (como estudiar inglés, programar, o hacer ejercicio), el sistema te otorga Puntos de Experiencia (XP) que se acumulan a lo largo del tiempo.

El sistema también cuenta con un catálogo de "Recompensas" (Shopping) que puedes canjear con tu XP ganado, fomentando un sistema de disciplina y gratificación equilibrado.

## ✨ Características

- **Múltiples Hábitos:** Registra misiones diarias de alto impacto como _Deep Work_, _Daily Coding_, _Master English_, _Strength Training_, etc.
- **Sistema de Puntos (XP):** Cada hábito tiene un valor asociado en puntos.
- **Guardado Persistente:** Tu XP de por vida se guarda en un archivo `config.json` local para que nunca pierdas tu progreso.
- **Tienda de Recompensas:** (En el código base) Un catálogo de recompensas, desde "Café Especial" (25 XP) hasta un "Fin de semana libre" (5000 XP) o un "Setup Upgrade" (2000 XP).

## 🛠️ Tecnologías

- Python 3
- Manejo de Archivos JSON (`json`, `os`)

## ⚙️ Cómo jugar

Simplemente ejecuta el archivo principal en la terminal. El programa te mostrará un menú numerado con los hábitos disponibles.

```bash
python3 Pincipal.py
```

Selecciona el número del hábito que completaste hoy y presiona `n` cuando termines para guardar tus puntos.
